# Chat Book Cloud (CBC) 前端 UI 设计与布局指南

## 一、设计理念 (Design Philosophy)

基于项目的核心价值观：“为创作者提效，为读者降噪”。本次前端设计的重塑，旨在打破传统博客平台信息过载的僵局，通过极简、专业和现代的设计语言，营造开发者的沉浸式体验空间。

*   **Sophisticated Minimalism (简约而不简单)**：剔除冗余干扰元素（坚决去广告化、去无关挂件），将屏幕核心空间留给“内容本身”和“创作者”。
*   **沉浸式与次世代体验**：适度结合毛玻璃 (Glassmorphism) 特效、平滑微动画，打造通透且富有空间层次感的现代 B 端/社区视觉体系。
*   **专业与极客感**：设计语言贴近 IDE (集成开发环境)，强化操作的高效性和所见即所得的掌控感。

## 二、全局视觉设计原子 (Design Tokens)

通过抽象出的变量管理项目的所有基础样式，确保多端和全局的一致性。

### 1. 色彩规范 (Color Palette)
*   **主色调 (Primary/Accent)**：推荐采用“极客蓝”或“科技青”（如 Tailwind 中的 `indigo-500` 或 `cyan-500`），作为动作触发（如发布、点赞激活）和链接的强调色。
*   **背景色 (Background)**：
    *   浅色主题：使用低饱和度的浅灰背景（如 `#F3F4F6`），拉开与纯白内容卡片的层次。
    *   深色主题（主推）：采用深灰或藏青作为底色（如 `#111827`），降低长时间阅读的视觉疲劳。
*   **界面层级透明度 (Glassmorphism)**：使用 `backdrop-filter: blur(12px)` 与带半透明通道的背景色 `rgba(255, 255, 255, 0.6)` 或黑夜模式的 `rgba(31, 41, 55, 0.7)` 相结合，替代厚重的实色边框区域。

### 2. 排版规范 (Typography)
*   **主界面字体**：使用现代化系统非衬线字体，如 `Inter`, `Roboto`, `PingFang SC`。
*   **阅读区与代码区**：正文行高控制在 `1.6` 以上保证易读性；代码块内容严格使用 `Fira Code`, `JetBrains Mono` 或 `Consolas` 等等宽字体，支持连字特性 (Ligatures)。

### 3. 微交互与动效 (Micro-interactions)
*   **卡片悬浮**：增加柔和的弥散阴影 `box-shadow` 并配合 `transform: translateY(-2px)` 的平滑上移，使用 `transition: all 0.3s ease`。

## 三、核心页面布局蓝图 (Layout Blueprints)

### 1. 沉浸式文章阅读页 (Article View)
**目标：高效降噪，让知识吸收路径最短。**
采用**典型的三栏布局**（参考经过市场验证的合理版式，但进行减法操作）：
*   **左侧功能挂件区 (Left Navigation)**：文章常用操作锚点（点赞、收藏、评论跳转、快速分享），随着页面滚动悬浮固定。
*   **中轴内容核心区 (Main Content)**：
    *   **AI 摘要层**：在文章标题和作者信息下方，优先且醒目地展示【AI Agent 生成的知识导图/极简解决方案】，给读者提供一分钟快问快答。
    *   **深阅读层**：正文内容去边框化，增加左右留白宽度，限制最大行字符数（建议 65-80 个字符/行）。
    *   **代码体验升级**：代码块需做到黑白高对比度，提供“一键复制”、“展开详情”功能，完全复刻本地 IDE 体验。
*   **右侧情报信息区 (Right Sidebar)**：精简版作者信息卡片、文章大纲结构 (TOC - 随页面向下滚动实时高亮当前进度)、无缝植入的高质量相关推荐。

### 2. IDE 级创作发文页 (Editor View)
**目标：创造无压力的书写环境。**
采用**全屏免打扰/双栏实时预览模式**：
*   **融合式顶部状态栏**：弱化边界感。仅保留核心的【发布按钮】、【环境设置】和至关重要的【实时保存进度 (Auto-saving... )】状态指示。
*   **混合动力编辑器主区**：
    *   使用 Markdown 核心语法兼容的富文本编辑器。
    *   **AI 协同区**：在编辑器侧边设立可伸缩的 AI Agent 面板，或者行内提示（如 Copilot 式的幽灵补全文字），实时呈现智能校对和内容预测建议。

### 3. 首页瀑布流与聚合专栏 (Home / Feed)
**目标：突出高质量内容分发。**
*   **全局顶栏 (Header)**：应用毛玻璃效果，提供高亮全局搜索框入口、用户状态及消息免打扰提醒口。
*   **核心流区**：
    *   弃用密集的信息流，改为网格化 (Grid) 或宽幅大卡片布局。
    *   卡片需明确透出作者标签、预计阅读时间、摘要片段及分类（通过颜色快速区分后端、前端、运维等模块）。

### 4. 社交与实时交流中心 (Chat & Social)
**目标：构建统一风格的站内互动桥梁。**
*   延续主站设计语言，经典的**左侧会话列表 + 右侧多功能聊天面板**布局。
*   在聊天过程中同样支持简短代码片段的高亮识别，对齐受众极客属性。
